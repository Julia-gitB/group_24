
Первое ДЗ 
Linux terminal (GitBash) commands

1) Посмотреть где я pwd
2) Создать папкy mkdir folder_name
3) Зайти в папку cd folder_name
4) Создать 3 папки в текущем катологе mkdir f_1 f_2 f_3            
5) Зайти в любоую папку cd f_1            
6) Создать 5 файлов (3 txt, 2 json) touch test1.txt test2.txt test3.txt test_1.json test_2.json
7) Создать 3 папки (вложенные друг в друга) mkdir -p d_1/d_2/d_3
8. Вывести список содержимого папки ls (выведет содержимое в строку) ls -la (выведет содержимое в столбик)
9) + Открыть любой txt файл vim test_1.txt
10) + написать туда что-нибудь, любой текст. Нажимаем букву i (insert) и пишем текст
11) + сохранить и выйти. Нажимаем по очереди esc:wq enter
12) Выйти из папки на уровень выше cd ..
—
13) переместить любые 2 файла, которые вы создали, в любую другую папку.  mv test_1.txt test_1.json d_1 (перемещение в директорию ниже)  
mv test1.txt test2.txt /d/Courses_QA (перемещение в директорию выше с указанием полного пути)
14) скопировать любые 2 файла, которые вы создали, в любую другую папку.  cp test2.txt test_2.json d_1
    или cp D:/Courses_QA/Terminal/f_1/test3.txt test2.txt D:/Courses_QA/Terminal/f_1/d_1/d_2
15) Найти файл по имени  find . -name test2.txt
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает. tail -f D:/Courses_QA/Terminal/f_1/d_1/test_1.txt
17) вывести несколько первых строк из текстового файла  head -n3 d_1/test_1.txt (выводит первые 3 строки)
18) вывести несколько последних строк из текстового файла  tail -n3 d_1/test_1.txt (выводит последние 3 строки)
19) просмотреть содержимое длинного файла (команда less) изучите как она работает. less /d/Courses_QA/Terminal/f_1/d_1/test_1.txt

20) вывести дату и время date (текущее дата и время),  date -r  /d/Courses_QA/Terminal/f_1/d_1/test_1.txt (дата и время последнего измениния файла)
